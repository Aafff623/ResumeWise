How to use

使用教程 · 演示流程

按下面步骤即可完整体验。业务功能在「智能诊断」页，不在本页表单操作。

1

进入智能优化

点击首页「开始智能优化」，进入填写页。左侧写/贴简历（Markdown），右侧选择目标岗位；可点岗位旁的「样板简历参考」对照模范写法。

2

编辑简历（Markdown）

默认编辑模式。可手写语法，或把 AI 生成的 Markdown 整段粘贴进来，再点「渲染预览」。不会语法可点标题栏右侧 ? 查看教学与最小示例预览。

3

选择岗位并开始诊断

单选 5 个 Mock 岗位之一，确认关键词与要求摘要后，点「开始 AI 诊断」。诊断依据 = 用户简历 + 岗位要求/关键词 + 样板简历。

4

查看诊断 → 生成优化版 → 导出

诊断页查看综合评分、匹配度、结构化问题；再生成优化版本，左右对照后可导出 MD / TXT / PDF / Word。

演示模式 vs 真实模式
演示模式 · Mock
  • 无需 API Key，开箱可跑通全流程
  • 诊断/优化结果为本地写好的示例数据
  • 适合答辩演示、断网、无额度场景
  • 顶栏默认选中此模式
真实模式 · DeepSeek
  • 需配置 DeepSeek API Key（仅存本机浏览器 localStorage）
  • 模型：V4 Flash（deepseek-v4-flash）/ V4 Pro（deepseek-v4-pro）
  • 需另开本机代理：cd resume-web && npm run proxy（:8787）
  • 诊断与优化结果由模型动态生成，随简历/岗位变化
  • A 同学的 Key 不会随仓库给到 B，需各自填写

切换方式：任意页顶栏「演示模式 · Mock / 真实模式 · DeepSeek」。仅真实模式显示「配置 API Key」。
