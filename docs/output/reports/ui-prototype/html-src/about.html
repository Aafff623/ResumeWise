项目定位

面向大学生求职场景的 AI 简历诊断、岗位匹配与优化演示系统（大二小学期）。完成闭环：

输入简历 → 选择目标岗位 → AI 诊断 → 展示问题 → 生成优化版本

技术栈
  • 前端：Vue3 · Vite · Element Plus · Axios（业务实现阶段）
  • 后端：Spring Boot 3 · REST · 无数据库
  • 数据：Mock JSON + 浏览器 localStorage
  • AI：仅后端调用；支持 AI Mock 模式
评分说明

综合评分 = 关键词匹配分 × 40% + AI 内容分 × 60%。关键词分本地计算，稳定可解释；AI 内容分由大模型评估表达与项目质量。

范围边界
  • 不做：登录注册、数据库、招聘爬虫、PDF/Word 解析、企业端
  • 不做：多轮聊天 Agent、自动投递
  • 必须：AI 经后端调用；Mock 模式可独立演示
